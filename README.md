## Описание проекта   

https://dummyapi.io/docs/models Данный сайт представляет собой сервис для тестирования API. Для выполнения запросов необходим app-id, который можно получить автоматически после регистрации на сайте. В качестве тестирования взят объект **Post**

## POST
___
### GET /post (Get List)
Возвращает список сообщений, отсортированных по дате создания.
- Доступны query params запроса для пагинации.
- Доступны созданные query params запроса.

**Response body**

**Post Preview**
```javascript
{
id: string(autogenerated)
text: string(length: 6-50, preview only)
image: string(url)
likes: number(init value: 0)
tags: array(string)
publishDate: string(autogenerated)
owner: object(User Preview)
}
```

**User Preview**
```javascript
{
id: string(autogenerated)
title: string("mr", "ms", "mrs", "miss", "dr", "")
firstName: string(length: 2-50)
lastName: string(length: 2-50)
picture: string(url)
}
```

### POST /post/create (Create Post)
Создать новую запись, вернуть данные о созданной записи.
Текст сообщения: Создание записи (поля владельца и записи обязательны для заполнения)

**Post Create**
```javascript
{
text: string(length: 6-50, preview only)
image: string(url)
likes: number(init value: 0)
tags: array(string)
owner: string(User id)
}
```

### PUT /post/:id (Update Post)
Обновить сообщение по идентификатору, вернуть обновленные данные сообщения.
Тело сообщения: Данные для отправки (поле владельца обновлять запрещено).

**Post**
```javascript
{
id: string(autogenerated)
text: string(length: 6-1000)
image: string(url)
likes: number(init value: 0)
link: string(url, length: 6-200)
tags: array(string)
publishDate: string(autogenerated)
owner: object(User Preview)
}
```
